## Core Concepts

LunivCore leverages the **performance benefits of C** while providing the **scripting flexibility of Lua** for a unique BI-as-Code experience.

<CardGrid stagger>
    <Card title="Business Intelligence As Code" icon="pencil">
        Adopt **Simple Syntax** for data operations, enabling robust **version control** and **unit testing** of your reports and dashboards.
    </Card>
    <Card title="Host Language (C)" icon="info">
        The C core efficiently **manages memory** and executes **computationally intensive tasks** for high-speed data processing.
    </Card>
    <Card title="Guest Language (Lua)" icon="document">
        Provides a **clean, dynamic scripting interface (DSL)** for users, which communicates with the C core via a defined, stable API.
    </Card>
    <Card title="The Glue: Lua C API" icon="right-arrow">
        This API acts as the secure, high-speed bridge, efficiently **passing data and commands** back and forth between the C and Lua environments.
    </Card>
</CardGrid>

## Installation For Devlopers

Requires git, build tools, and a C compiler.

<Steps>

1. Use your terminal:

	```bash
	git clone https://github.com/Sieep-Coding/LunivCore
	```

2. `cd` into the source code:
	```bash
	cd LunivCore
	```
3. Run the installation command
	```bash
	make
	```
4. Run the installation command
	```bash
	./lcore examples/helloworld.lcore
	```
</Steps>
